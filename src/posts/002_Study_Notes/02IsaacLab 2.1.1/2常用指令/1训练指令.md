# 官方和外部项目训练常用指令笔记

## 1. 说明

下面整理的是在 Isaac Lab 官方项目和你自己的外部项目中，最常用的一些指令，包括：

- 随机策略运行
- 正式训练
- 回放
- 回放并录制视频
- 查看训练曲线
- 查看显卡使用率
- 查找模型文件

为了避免混淆，下面分成两部分：

1. **官方 Isaac Lab 项目常用指令**
2. **你自己的外部项目 `ur10e_ocs_2` 常用指令**

---

## 2. 进入对应根目录并且激活虚拟环境

### 2.1 官方 Isaac Lab 项目

先进入 Isaac Lab 根目录：

```bash
cd /home/isst/IsaacLab
```

### 2.2 外部项目 `ur10e_ocs_2`

先进入外部项目目录：

```bash
cd /home/isst/Action/04lab_example/ur10e_ocs_2
```

### 2.3 激活 Isaac Lab 虚拟环境

先进入项目根目录：

```bash
conda activate env_isaaclab
```

---

## 3. 随机策略运行

### 3.1 官方 Isaac Lab 项目

运行官方 UR10 Reach 任务的随机策略：

```bash
./isaaclab.sh -p scripts/random_agent.py --task Isaac-Reach-UR10-v0 --num_envs 64
```

---

### 3.2 外部项目 `ur10e_ocs_2`

运行外部项目的随机策略：

```bash
python scripts/random_agent.py --task Template-Ur10e-Ocs-2-v0 --num_envs 64
```

---

## 4. 正式训练

### 4.1 官方 Isaac Lab 项目

普通训练：

```bash
./isaaclab.sh -p scripts/reinforcement_learning/skrl/train.py --task Isaac-Reach-UR10-v0 --num_envs 64
```

无头训练（推荐）：

```bash
./isaaclab.sh -p scripts/reinforcement_learning/skrl/train.py --task Isaac-Reach-UR10-v0 --num_envs 256 --headless
```

更大并行环境数训练：

```bash
./isaaclab.sh -p scripts/reinforcement_learning/skrl/train.py --task Isaac-Reach-UR10-v0 --num_envs 1024 --headless
```

---

### 4.2 外部项目 `ur10e_ocs_2`

普通训练：

```bash
python scripts/skrl/train.py --task Template-Ur10e-Ocs-2-v0 --num_envs 64
```

无头训练：

```bash
python scripts/skrl/train.py --task Template-Ur10e-Ocs-2-v0 --num_envs 256 --headless
```

更大并行环境数训练：

```bash
python scripts/skrl/train.py --task Template-Ur10e-Ocs-2-v0 --num_envs 1024 --headless
```

## 5. 查看回放

### 5.1 官方 Isaac Lab 项目

先查找训练好的模型：

```bash
find logs/skrl -name "best_agent.pt"
```

回放官方 UR10 任务：

```bash
./isaaclab.sh -p scripts/reinforcement_learning/skrl/play.py --task Isaac-Reach-UR10-Play-v0 --num_envs 1 --checkpoint 你的best_agent.pt路径
```

---

### 5.2 外部项目 `ur10e_ocs_2`

先查找训练好的模型：

```bash
find logs/skrl -name "best_agent.pt"
```

回放外部项目任务：

```bash
python scripts/skrl/play.py --task Template-Ur10e-Ocs-2-v0 --num_envs 1 --checkpoint 你的best_agent.pt路径
```

---

## 6. 查看回放并录制视频

先确保安装了 `ffmpeg`：

```bash
sudo apt update
sudo apt install ffmpeg
```

### 6.1 官方 Isaac Lab 项目

先查找训练好的模型：

```bash
find logs/skrl -name "best_agent.pt"
```

回放并录制视频：

```bash
./isaaclab.sh -p scripts/reinforcement_learning/skrl/play.py --task Isaac-Reach-UR10-Play-v0 --num_envs 1 --checkpoint 你的best_agent.pt路径 --enable_cameras --video --video_length 600
```

录制完成后查找视频：

```bash
find logs -type f \( -name "*.mp4" -o -name "*.gif" \)
```

---

## 8. 查看显卡使用率

在任意目录都可以执行。

实时查看显卡状态：

```bash
watch -n 1 nvidia-smi
```

查看 GPU 利用率、已用显存、总显存：

```bash
nvidia-smi --query-gpu=utilization.gpu,memory.used,memory.total --format=csv -l 1
```

---

## 9. 查看任务名称

### 9.1 官方 Isaac Lab 项目

查看所有任务名称：

```bash
./isaaclab.sh -p -c "import gymnasium as gym; import isaaclab_tasks; print(sorted(gym.registry.keys()))"
```

只查看和 UR10、Reach 相关的任务名称：

```bash
./isaaclab.sh -p -c "import gymnasium as gym; import isaaclab_tasks; print([k for k in sorted(gym.registry.keys()) if 'UR10' in k or 'Reach' in k])"
```

---

### 9.2 外部项目 `ur10e_ocs_2`

如果你的外部项目已经正确安装并能被当前环境识别，也可以类似查看。  
不过通常你自己的任务名已经固定为：

```text
Template-Ur10e-Ocs-2-v0
```

如果只是日常使用，直接记住这个任务名即可。
